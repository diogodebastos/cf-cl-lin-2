# Update CV on LinkedIn
## Purpose
Update the user's CV and profile information on LinkedIn via www.linkedin.com. This can include changes to work experience, education, skills, or any other relevant sections of the LinkedIn profile. Ensure that the updates are accurate and reflect the user's current professional status. After updating the CV, confirm that the changes have been successfully saved on LinkedIn.
The base CV information is stored in data/cv_data/. Use these files as the source of truth for the CV updates on LinkedIn. But you can also edit this information, as long as you NEVER LIE. This will be usefull to teak the cv to the job you are applying for. Always make sure the CV is up to date with the latest information from data/cv_data/.cv.md, data/cv_data/work_experience.md, data/cv_data/research.md, data/cv_data/education.md, data/cv_data/publications.md, data/cv_data/skills.md, and data/cv_data/conferences.md.
