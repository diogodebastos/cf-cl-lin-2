# Send LinkedIn Connection Invitation
## Purpose
Send a connection invitation to a LinkedIn user via www.linkedin.com.

## Message Content
The message content for the connection invitation should be a personalized note that introduces yourself and explains why you want to connect with the recipient. The message should be concise and professional, highlighting any common interests or connections that may help establish a rapport with the recipient. Always explain you are like: "I'm actually writing this with help from Claude, an AI assistant that is navigating LinkedIn on my behalf."
