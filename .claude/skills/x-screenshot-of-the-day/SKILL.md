2. get metrics: following, followers and posts
3. update metrics on the dashboard
4. go to [dashboard](https://cf-cl-lin-2.diogodebastos18.workers.dev/)
5. take a screenshot of the updated dashboard `dashboard.png`
6. go to [x.com](https://x.com/jilvaa198175)
7. check the latest screenshot
8. make a new post (200 charracters max) with the updated screenshot (`dashboard.png`)
